开源量化框架

调研精选：可与本平台互补的开源策略库 / 交易框架。当前以导航与集成路线为主，不内嵌整套 bot。

与本平台怎么配合： Crypto Analyst 专注 AI 分析、规则盯盘、周期位置与经典回测；自动下单仍规划中。 更合适的集成方式是「外挂执行引擎」：本平台出信号 / 告警，Freqtrade / Jesse 等负责回测优化与 dry-run。 许可证注意：Freqtrade 为 GPL-3.0，若深度链接需评估合规；Jesse / Hummingbot / vectorbt / Lean 多为 MIT 或 Apache-2.0，更易做适配层。

推荐集成优先级

建议集成路线

  • ✅ 本平台：盯盘告警 · 策略库 · 经典回测 · 周期工具
  • ⏳ 阶段 A：策略信号导出 JSON / Webhook，对接 Freqtrade dry-run
  • ⏳ 阶段 B：可选 sidecar 运行 Jesse / Freqtrade，结果回写告警面板
  • ⏳ 阶段 C：研究侧用 vectorbt 做参数扫描，优胜参数回灌本平台回测
  • ⏸ Hummingbot：做市 / 套利场景，与当前「提醒型」定位不同，暂不优先
  • ⏸ NautilusTrader / Lean：机构级事件驱动，复杂度高，长期可选
